\vspace{-2cm}
\Large
Perform the following hexadecimal additions:
\begin{enumerate}
\item $(AF5)_{16} + (33C)_{16}$
\item $(A5E)_{16} + (B21)_{16}$
\item $(53B)_{16} + (FA7)_{16}$
\item $(BDE)_{16} + (A3C)_{16}$
\end{enumerate}
